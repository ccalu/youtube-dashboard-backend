# Scripts SQL

Esta pasta contém scripts SQL para manutenção e atualização do banco de dados.

## Organização:
- Scripts de criação de tabelas
- Scripts de alteração de estrutura
- Scripts de correção de dados
- Queries úteis para análise

## Uso:
Scripts SQL devem ser executados diretamente no Supabase SQL Editor.

## Importante:
- Sempre fazer backup antes de executar scripts em produção
- Documentar o propósito de cada script
- Scripts aplicados podem ser arquivados ou deletados
