Dashboard de Analise de Conteudo

Analise semanal de performance dos nossos canais e videos
19/02/2026 · Marcelo + Micha

A Ideia

Um dashboard web que toda segunda-feira gera um relatorio de performance dos nossos videos. Mostra as metricas de cada canal e cada video, cruza com as planilhas de producao (titulo, copy, estrutura usada) e monta uma visao clara do que esta funcionando.

So entram videos da nova fase — videos que tem a coluna A preenchida na planilha. Videos antigos sao ignorados.

O Cerebro do Sistema

Cada relatorio semanal é salvo no banco. O sistema sempre analisa a semana atual + todas as analises anteriores. É um cerebro que se auto-alimenta — quanto mais semanas, mais inteligente.

Semana 1: Primeiros dados, analise inicial
Semana 2: Analise da semana + compara com a anterior
Semana 5: Tendencias por canal comecam a aparecer
Semana 10+: Cerebro robusto — sabe o que funciona em cada canal
Mais semanas = mais dados = analises mais precisas

Fonte dos Dados

Planilhas de Producao (Google Sheets - 1 por canal)

Col AEstrutura de copy/titulo (FILTRO PRINCIPAL) Col BTitulo do video Col CDescricao do video Col DScript / Copy do video Col EStatus (produzido ou "to do")
So entram videos com coluna A preenchida = nova fase de producao.

Banco de Dados (Supabase - coleta automatica diaria)

  • Views, likes, comentarios (snapshots diarios)
  • Inscritos e crescimento
  • 43 canais monitorados

YouTube Analytics API (precisa ativar)

  • CTR — % que clicou depois de ver o titulo/thumb
  • Retencao — % do video que as pessoas assistem
  • Watch time, fontes de trafego

Precisa re-autorizar OAuth dos canais (adicionar 1 escopo)

Decisoes do Micha

Pra comecar a construir:

1. Criterios de Filtro

Views minimas pra um video entrar no relatorio?
Pode variar por subnicho (ex: Monetizados = 10k, Guerra = 5k)
Quantos dias minimo desde a publicacao?
Sugestao: 7 dias (tempo pro video performar)

2. Coluna A - Estruturas

Quais valores vao aparecer na coluna A?
Ex: A1, A2, B1, C3... o que cada letra/numero representa?

3. Planilhas

Compartilhar os IDs das planilhas de cada canal
Formato: Col A = estrutura, B = titulo, C = descricao, D = script, E = status

4. Para o futuro (aba Analise)

Como voce analisa copy e titulo hoje? Quais seus criterios?
Isso vai alimentar a LLM quando ativarmos a aba de analise
Relatorio toda segunda-feira · Salvo no banco · Cerebro acumulativo
