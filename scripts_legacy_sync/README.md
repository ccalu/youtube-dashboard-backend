# Scripts Legados - Backup

Estes scripts foram substituídos pelos novos scripts Python universais que funcionam em qualquer sistema sem limitações.

## Scripts Antigos (backup):
- `sync.bat` - Sincronização para Windows
- `sync.sh` - Sincronização para Mac/Linux
- `setup.bat` - Setup para Windows
- `setup.sh` - Setup para Mac/Linux

## Use os Novos Scripts Python:
```bash
# Para sincronizar (funciona em qualquer sistema)
python sync.py

# Para configurar (funciona em qualquer sistema)
python setup.py
```
