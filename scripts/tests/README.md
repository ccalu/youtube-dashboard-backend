# Scripts de Teste

Esta pasta contém scripts de teste temporários.

## Uso:
```bash
python scripts/tests/test_exemplo.py
```

## Importante:
- Esta pasta é para testes TEMPORÁRIOS
- Limpar periodicamente (remover testes antigos)
- Testes permanentes devem usar pytest
- Não committar testes desnecessários

## Limpeza:
Revisar semanalmente e deletar testes que não são mais úteis.
